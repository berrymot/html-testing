how to do that ↑

give your table cells a class of r and surround their contents with a span, then add this css:

td {
  border: 1px solid black; /* optional */
}
:is(th, td).r {
  transform: rotate(180deg);
}
:is(th, td).r span {
  writing-mode: vertical-rl; /* -lr also works unless you want linebreaks to work properly */
}

notes and drawbacks

  • does not work on mobile
  • may subtly affect backgrounds and borders
  • you could in theory make it work for 45° angles too? maybe? haven't tried yet
