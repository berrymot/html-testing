rotated not rotated

how to do that ↑

<style>
  td {
    border: 1px solid black; /* optional */
  }
  :is(th, td).r {
    transform: rotate(180deg);
  }
  :is(th, td).r span {
    writing-mode: vertical-rl; /* -lr also works unless you want linebreaks to work properly */
  }
</style>

<table>
  <tr>
    <td class=r><span>rotated</span></td>
    <td>not rotated</td>
  </tr>
</table>

notes and drawbacks

  • does not work on mobile
  • may subtly affect backgrounds and borders
  • you could in theory make it work for 45° angles too? maybe?
