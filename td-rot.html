give your table cells a class of r and surround their contents with a span ...

<th class=r><span>stuff</span></th>

... and then add this css.

:is(th, td).r {
  transform: rotate(180deg);
}
:is(th, td).r span {
  writing-mode: vertical-rl; /* `vertical-lr` also works unless you want linebreaks to work properly */
}
tr:first-child th.r {
  vertical-align: top; /* yes, `top` (because it's rotated) */
}

notes and drawbacks

  • does not work on mobile - the cells become really narrow and the text overflows to the left
  • may subtly affect backgrounds and borders
  • lines don't wrap unless you add explicit linebreaks (or set a max-height maybe)
